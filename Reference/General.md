# 智能机械臂控制软件 - 项目需求说明文档 (General Context)

## 1. 项目概述
本项目要求开发一套基于 **MFC (Microsoft Foundation Classes)** 的桌面应用程序，用于远程控制一个带有摄像头的 **6 自由度智能机械臂**。该软件需要实现人机交互、实时视频流显示、串口指令通讯以及基础的“智能”反馈逻辑。

## 2. 核心开发目标
1. **良好的人机交互界面**：基于 MFC 对话框 (Dialog-based) 开发，布局直观。
2. **摄像头视频采集**：实时获取机械臂自带摄像头的画面并显示在 UI 界面上。
3. **串口通讯 (Serial Port)**：通过串口向机械臂发送控制指令，驱动 6 个舵机动作。
4. **智能体现**：实现自主逻辑（例如：颜色追踪、自动避障或特定手势动作组合），具体逻辑需基于原生算法实现。

## 3. 硬件规格 (机械臂结构)
根据项目图示，机械臂包含 **6 个舵机** 和 **1 个摄像头**：
- **舵机 1-6**：分别负责机械臂的底座旋转、关节升降、手爪开合等自由度。
- **摄像头**：安装于机械臂顶端，用于获取第一视角画面。

## 4. 关键限制条件 (重要)
- **尽量减少对第三方插件的依赖**：尽量不使用太怪异的库，视觉处理部分除外。
- **原生开发**：所有图像处理和底层逻辑都尽量使用原生 C++/MFC 代码实现。
