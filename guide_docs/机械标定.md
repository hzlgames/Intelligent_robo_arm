# 机械参数标定指南

为了让机械臂能准确计算“末端在空间中的位置”（运动学解算），你需要用直尺或游标卡尺测量以下 5 个关键物理长度。

## 1. 测量示意图 (对应舵机ID)

请对照 `Reference/the_arm.png` 中的舵机编号进行测量。

> **测量原则**：所有长度均为两个**旋转轴中心点**之间的直线距离。

## 2. 待测参数表

| 参数符号 | 对应物理结构 | 测量起点 (旋转轴) | 测量终点 (旋转轴) | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **$L_{base}$** | **底座高度** | 地面 / 底板 | **舵机 5** (肩部) | 机械臂“肩膀”离地面的高度 |
| **$L_{arm1}$** | **大臂长度** | **舵机 5** (肩部) | **舵机 4** (肘部) | 两个大扭力舵机之间的臂长 |
| **$L_{arm2}$** | **小臂长度** | **舵机 4** (肘部) | **舵机 3** (腕部) | 肘部到手腕旋转轴的距离 |
| **$L_{wrist}$** | **腕部长度** | **舵机 3** (腕部) | **舵机 2** (云台底座) | 注意：是从腕部摇摆轴到云台水平旋转面的距离 |
| **$L_{cam}$** | **末端偏移** | **舵机 1** (摄像头俯仰) | 摄像头镜头中心 | 决定了“眼睛”实际在看哪里 |

## 3. 为什么需要这些？

- **$L_{arm1}$** 和 **$L_{arm2}$** 决定了机械臂能伸多远。
- **$L_{base}$** 决定了机械臂的工作高度平面。
- **$L_{cam}$** 对于视觉抓取至关重要，因为摄像头看到的物体坐标是以摄像头为原点的，我们需要把它转换回机械臂基座坐标系。

## 4. 记录方式

建议直接用纸笔记录下来，后续我们会添加到软件的配置页面中：

```text
L_base  = ____ mm
L_arm1  = ____ mm
L_arm2  = ____ mm
L_wrist = ____ mm
L_cam   = ____ mm
```
